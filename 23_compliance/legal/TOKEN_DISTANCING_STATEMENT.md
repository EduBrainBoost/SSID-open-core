# SSID-open-core Token Distancing Statement

## Erklärung
SSID-open-core ist eine Open-Source-Bibliothek für Self-Sovereign Identity.

Dieses Repository hat **KEINE Verbindung** zu:
- Fremden SSID-Token auf Binance Smart Chain (BSC/BEP-20)
- Token-Listings auf CoinGecko, CoinMarketCap oder anderen Plattformen
- SSID-Token-Airdrops, DAO-Governance oder NFT-Projekten Dritter
- Dem WLAN-Begriff "SSID" (Service Set Identifier)
- SSI Digital oder anderen gleichnamigen Projekten

## Verbindliche Regeln
- Kein Code in diesem Repository darf fremde Token-Claims integrieren
- Keine Werbung oder Listung fremder Tokens
- Keine Custody-, Airdrop- oder DAO-Implikationen
- Keine Verwechslung mit BSC-basierten SSID-Tokens

## Kanonischer Status
Der SSID-Token des SSID-Projekts ist ausschließlich:
- Utility (Zugang, Governance, Rewards)
- Non-custodial
- Hash-only (keine PII on-chain)
- MiCA-konform (kein Zahlungsmittel)

## Durchsetzung
Diese Abgrenzung wird durch Policy-Dateien und Tests in diesem Repository prüfbar gemacht.
