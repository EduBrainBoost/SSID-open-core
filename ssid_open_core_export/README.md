# SSID Open-Core Public Export

This is the canonical public export of SSID Open-Core modules.

## Included Modules

- 03_core: Smart Contracts & Dispatcher logic
- 12_tooling: CLI tools, SDKs, and automation scripts
- 16_codex: Knowledge base, policies, and system definitions
- 23_compliance: Regulatory compliance and audit evidence
- 24_meta_orchestration: Orchestration, registry, and coordination

## Usage

1. Extract this archive
2. Review EXPORT_MANIFEST.json for file listing
3. Verify SHA256 hashes: `sha256sum -c export.sha256.txt`
4. Integrate modules as external dependencies in your project
